PYTHON BASICS

INTRODUCTION

Python is a programming language used mainly for projects that include working with files, which contain numbers or lists, among other uses. Some examples of these could be:

  • Files that save balance sheets
  • Data sheets with historical events numbers

This is extremely useful when you want to study numbers from past events.

CREATING VARIABLES

Variables in Python can be from many different types. Here are the three most common types of them and some examples.

  • String
  • Int
  • Float
name = "Maradona" #string
age = 63 #int
height = 1.65 #float

This are the 3 most used ones, you can find many more types like booleans, among others.

PRINTS

When you declare a variable, the computer stores it's value. You can call it for it to be shown on screen at any time by using prints. The uses for prints are usually the following:

  • Showing the result of a code
  • Giving updates as the code is working
  • Debugging
x = 10
y = 5
print(x + y)

You can also do maths or formulate phrases inside prints, by following it's sintaxis.

DEBUGGING

As a Python Developer, your code will encounter bugs very often. This can be caused by many different errors:

  • Sintaxis error
  • Undeclared variables
  • Missing libraries
  • Broken codes
x = 10
while x>10
x = x + 1

The code presented has many errors, like missing indentation, code that makes no sense and wrong sintaxis. A way to debug it could be adding prints among all the code to follow it step-by-step.

ELSE AND IF

The most common type of conditionals are else and if. These have the following functions:

  • IF: checks if the condition given is true or false, and works based on the answer.
  • ELSE: followed by an if, adds another condition in case the first one isn't true.
x = int(input())
if x>5:
print("is bigger than 5")
else:
print("is smaller or equal to 5")

You can make it longer by adding ELIF's to the code. Here is an example following the previous code:

x = int(input())
if (x>5):
print("is bigger than 5")
elif(x=5):
print("is equal to 5")
else():
print("is smaller than 5")

This is the type of functions you will be using the most through your journey as a Python Developer, so you better get on well with them.
